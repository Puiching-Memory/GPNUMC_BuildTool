# GPNUMC_BuildTool

GPNU[广东技术师范大学]MineCraft 建筑工具

# 功能

* [X] .OBJ文件自动转换为.schem文件，这可以被创世神(EditWorld mod)载入

# 结构

analyze.py:导入和分析.OBJ文件

output.py:生成和导出.schem文件

# 所需库

* python3.11
* mcschematic
* numpy

# OBJ-3D软件导出

## 1.blender

导出OBJ,坐标轴选择Y/Z
